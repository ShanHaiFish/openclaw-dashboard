# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.1.0] - 2026-03-22

### Added
- 🐙 GitHub skill integration with real API
- 📊 Token cost analytics with model pricing
- 🔄 Auto-refresh for skills page (30s interval)
- 🌐 Real-time WebSocket data updates
- 📱 Enhanced mobile navigation

### Fixed
- Agents page data consistency (6 agents displayed)
- Analytics page agent count synchronization
- Skills page YAML frontmatter parsing
- API data format compatibility issues

## [2.0.0] - 2026-03-21

### Added
- 🎨 Complete UI redesign with glass morphism
- 🌙 AMOLED theme support
- ⌘K Command Palette
- 🌐 Full i18n (Chinese/English)
- 📊 Analytics dashboard with token tracking
- 🔔 Alert center
- ⚡ Activity feed
- 🧩 Skills visualization
- ⌨️ Keyboard shortcuts system
- 🐳 Docker support with multi-stage build
- 🔄 GitHub Actions CI/CD

### Technical
- React 18 + Vite 5
- TailwindCSS 3
- Framer Motion animations
- Recharts for data visualization
- Zustand state management
- WebSocket real-time updates

### Added
- Initial release
