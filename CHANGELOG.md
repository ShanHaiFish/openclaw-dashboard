# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-22

### Added
- 🎨 Glass morphism UI with 4 themes (Dark/Light/AMOLED/System)
- ⌘K Command Palette for quick navigation
- 🌐 Full i18n support (Chinese/English)
- 📊 Real-time dashboard with WebSocket updates
- 📱 Fully responsive design (mobile/tablet/desktop)
- 🔔 Notification system with activity feed
- 🧩 Plugin management and visualization
- ⌨️ Keyboard shortcuts system
- 🐳 Docker support with multi-stage build
- 🔄 GitHub Actions CI/CD
- 📈 Analytics dashboard with token tracking
- 📋 Session and Agent management views

### Technical
- React 18 + Vite 5
- TailwindCSS 3 + CSS Variables theming
- Framer Motion animations
- Recharts for data visualization
- Zustand state management
- Express.js backend with WebSocket
- ESLint + Vitest configured
