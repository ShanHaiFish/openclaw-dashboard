### 主要界面预览

1. **首页概览** - 6 个统计卡片 + 实时数据
2. **会话管理** - 会话列表 + 详情面板
3. **Agent 管理** - Agent 网格 + 性能图表
4. **分析面板** - Token 使用趋势 + 成本分析
5. **技能页面** - 技能列表 + 状态监控
6. **设置页面** - 主题切换 + 语言设置
